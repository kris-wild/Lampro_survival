get_url

extracts url from mydata

Contents

  • Syntax
  • Description
  • Remarks
  • Example of use

Syntax

url = get_url(my_pet, bibkey)

Description

Extracts url from a bibkey in metaData.biblist in a mydata-file

Input

  • character string with name of my pet
  • optional character string with bibkey (default 'Wiki' or 'wiki')

Output

  • character string with url

Remarks

bibkey should be a field of metaData.biblist and contain http, else the result is empty. If several http's exist in a bib-entry, only the first one is extracted

Example of use

get_url('Ciona_intestinalis') or get_url('Salvelinus_alpinus', 'fishbase') or get_url('Raja_clavata', 'Kooy2010')

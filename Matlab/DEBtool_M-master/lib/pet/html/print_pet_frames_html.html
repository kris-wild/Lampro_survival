Syntax

print_pet_frames_html (metadata)

Description

Prints four html pages for each entry :

  • i_my_pet.html
  • i_results_my_pet.html
  • i_left_my_pet.html
  • i_top_my_pet.html

Input:

  • metadata: structure

Remarks

Keep in mind that this function is specifically designed for created the webpage of add-my-pet - this function is called when local directory is html and when we go one level up it supposes the existance of a subdirectory called results ! see

load('results_my_pet.mat'); print_pet_frames_html(metadata)

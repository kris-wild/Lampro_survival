betaML

Calculates ML estimates for the parameters of the beta distribution

Syntax

[par_ML, info] = betaML (x)

Description

Calculates ML estimates for parameters alpha and beta for the beta distribution: pdf(x)=x^(alpha-1)(1-x)^(beta-1)/ B(alpha, beta); df(x)=I_x(alpha, beta).

Input:

  • x: vector with trials from a beta distribution

Output:

  • par_ML: 2-vector with ML estimates for the parameters alpha and beta
  • info: boolean for success (1) or failure (0)

Remarks

generate random trials from a beta distribution with betarnd; dim(alpha) = 0 and dim(beta) = 0.

Example of use

betaML(betarnd(.2,.4,100,1))

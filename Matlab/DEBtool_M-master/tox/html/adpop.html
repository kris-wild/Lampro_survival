adpop

adaptation effects on population growth (of micro-organisms)

Contents

  • Syntax
  • Description
  • Example of use

Syntax

f = adpop(p, t, c)

Description

adaptation effects on population growth (of micro-organisms) toxico kinetics is instantaneous hazard rate linear in (internal) concentration during initial time increment only

Input

  • p: (4,k) matrix with parameters values in p(:,1) (see below)
  • t: (nt,1) matrix with exposure times
  • c: (nc,1) matrix with concentrations of toxic compound

Output

  • f: (nt,nc) matrix with population size

Example of use

see <../mydata_pop.m *mydata_pop*>

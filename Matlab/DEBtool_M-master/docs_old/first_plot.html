Plotting

Plots are produced as follows:

x = linspace(0,10,40); % create vector of values of independent variable
y = x + 0.1*x.^2; % obtain vector with corresponding dependent variable
plot(x,y,'g'); # plot y against x, connecting points in green
xlabel('independent variable'); % set text along x-axis
ylabel('dependent variable'); % set text along y-axis

Set x-range to (0,10) and y-range to (1,2) with axis([0 10], [1 2]). Reset range specifications with axis. Close plots with clf or close. Create multiple plots like:

z = 1./y; % create second dependent variable
clf % clear previous plot
subplot(1,2,1); % first subplot in 1 row, 2 colums
xlabel('independent variable'); % set text along x-axis
ylabel('dependent variable 1'); % set text along y-axis
plot(x,y,'g'); % plot in green

subplot(1,2,2); % second subplot in 1 row, 2 colums
xlabel('independent variable'); % set text along x-axis
ylabel('dependent variable 2'); % set text along y-axis
plot(x,z,'r'); % plot in red

Combine both curves in a single plot with

clf % clear previous plot; suppress legends
xlabel('independent variable');
ylabel('dependent variables');
plot(x,y,'g',x,z,'r'); % plot y against x in green and z gainst x in red

Alternatively, add more to an existing plot with hold on;.

clf; hold on; plot(x,y,'g'); plot(x,y,'.r'); # curve in green, dots in red

Write plot to encapsulated postscript file nm.eps with

print -deps nm.eps

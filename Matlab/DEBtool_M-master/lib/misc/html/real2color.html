real2color

gets RGB colours for a vector of reals using lava colour coding

Contents

  • Syntax
  • Description
  • Remarks
  • Example of use

Syntax

[RGB range] = real2color(x, r)

Description

Obtains RBG colors from reals using lava-colour coding (black, blue, magenta, red, white) The smallest value gets black, the largest almost white.

Input

  • x: n-vector real number
  • r: optional scalar with range factor > 1 (default 1.1)

Output

  • (n,3)-matrix with RGB colours
  • range: 2-vector with x-values that correspond with black and white

Remarks

For r = 1, the largest value of x gets white. Uses linear interpolation between colours

Example of use

RGB = real2color(1:10);

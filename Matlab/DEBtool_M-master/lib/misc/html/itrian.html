itrian

wraps the upper triangular elements of an upper triangular matrix into a vector

Contents

  • Syntax
  • Description
  • Remarks
  • Example of use

Syntax

r = itrian (m)

Description

Wraps the upper triangular elements of an upper triangular matrix into a vector; This function is inverse to 'trian'

Input:

  • m: (n,n)-matric

Output:

  • r: vector with elements of length n * (n - 1)/ 2

Remarks

Function trian is inverse to function itrian.

Example of use

trian([2 4 7]) and itrian(trian([2 4 7]))

f_ris0_hex

Gets scaled functional response at with the specific population growth rate is zero for the hex model

Contents

  • Syntax
  • Description
  • Remarks

Syntax

[f, info] = f_ris0_hex (par)

Description

Obtains the scaled function response at which specific population growth rate for the hex model equals zero, by solving the characteristic equation. We work at T_ref here.

Input

  • par: structure with parameters for individual

Output

  • f: scaled func response at which r = 0 for the hex model
  • info: scalar with indicator for failure (0) or success (1)

Remarks

optional thinning (a boolean, default 1), and background hazards h_B0b, h_Bbj, h_Bje, h_Bei (default all 0) must be added to par before use, if necessary. par.reprodCode must exist. Only imago's lay eggs, but do not allocate to reprod, at a rate of reprod buffer at j per expected life time as imago, till buffer is finished.

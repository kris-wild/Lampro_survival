Syntax

prt_mydata (data, auxData, metaData, txtData)

Description

Writes file mydata_my_pet.m from 4 data-structures; it is inverse to mydata_my_pet.m.

Input:

  • data: structure with data
  • auxData: structure with auxilirary data
  • metaData: structure with metaData
  • txtData: structure with text for data

Remarks

The file will be saved in your local directory; use the cd command to the dir of your choice BEFORE running this function to save files in the desired place. All weights are set at default values in the resulting file; you might want to change them in the file during the estimation process. This version ignores auxData other than temperature.

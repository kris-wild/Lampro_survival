surv_f

Calculates survivor function of F-distribution

Contents

  • Syntax
  • Description
  • Remarks
  • Example of use

Syntax

Q = surv_f (nu, F)

Description

Calculates survivor function of F-distribution

Input:

  • nu: (2,1)-vector with degrees of freedom (integers)
  • F: (n,1)-vector of argument values

Output:

  • Q: (n,1)-vector with survivor probabilities

Remarks

Abramowitz & Segun 1965, {946} 26.6.6-8; Warning: does not work properly for v1 and v2 odd while v1 > 1 !!!!! This function is inverse to survi_f.

Example of use

surv_f([3;2], [1 2 3])


Published with MATLAB® 7.14

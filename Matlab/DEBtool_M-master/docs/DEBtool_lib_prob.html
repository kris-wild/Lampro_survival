DEBtool Toolbox: lib_prob

Routines that relate directly or indirectly to probability calculus.

Survivor functions

Function surv gives plot-cordinates of an empirical survivor function; frac calculates empirical survivor probabilities. Function lik2surv calculates the survivor function from the -2 log lik ratio function; haz2surv does that from the hazard rate;

Function surv_chicalculates the survivor probability of the Chi-square distribution; surv_f does so for the F-distribution, with survi_f is its inverse.

Function surv_count obtains Monte Carlo counts for the number of surviving subjects in time.

Means

Function fnchd obtains the expected value of Fisher's non-central hypergeometric distribution. Function ellipse gets cordinates of isoclines of the bivariate normal pdf.

Function bincoeff gets binomial coefficients.

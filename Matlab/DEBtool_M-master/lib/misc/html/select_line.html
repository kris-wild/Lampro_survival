Syntax

line = select_line (line)

Description

Selects type, linewidth, color of a line

Input:

  • line: optional 3-vector of cells with type, linewidth, color of a line

Output:

  • marker: 3-vector of cells with type, linewidth, color of a line

Remarks

Press OK when done

Example of use

line = select_line;

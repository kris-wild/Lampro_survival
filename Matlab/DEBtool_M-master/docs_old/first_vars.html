Variables

Create a variable by assigning a value to it using symbol =. Show the value on screen by typing its name, or suppressing the character ;. Elementary operators are +, -, *, /, ^. Try:
x = 2; % value is not shown because of ;
x % show value of x
y = 3 % assign and show value of y
x+y % show result of addition
y = "text string" % fill a variable with text

A scalar is an object that has the value of a number. The power of Octave is the easy of working with a matrix (a 2-dimensional array with numbers) or a higher-dimensional array; a vector is a special matrix with only one row or column. Create a matrix with [numbers-separated by ',' and/or ' '], the character ; indicates a new row. Get an element of a matrix by bracketing the indices with (,) and a full row with (,:). The command [] creates the empty matrix. Try:
x = [2 5 3;1 6 4] % create a (2,3)-matrix
x(2,2) % show element with index 2,2
x(1,:) % show first row
[x,[1;3]] % catenate an extra column to x
[x;[1 3 5]] % catenate an extra row to x
x(:,2) = [] % delete column 2
x(1,1) = 3 % change element (1,1) into 3
y = ['example'; 'trial'] % spaces are added automatically

Obtain the size of an existing matrix with size in two ways:
nx = size(x) % the variable nx is now a (1,2)-matrix
[r,c] = size(x) % the variables r and c are both (1,1)-matrices
r = size(x,1) % the variable r is a (1,1)-matrix
c = size(x,2) % the variables c is a (1,1)-matrix

The elements of a vector can be assessed by using 1 or 2 indices. The number of elements can be obtained with length. Create special vectors with
x = 0:5 % this creates a row-matrix from 0 to 5
x(4) % show element with index 4
x(1,4) % another way to show that element
x([4 2 1]) % show elements with indices 4 2 and 1
x = 0:0.5:3 % row-matrix from 0 to 3 with steps size 0.5
length(x) % show the number of elements
x = linspace(3,4,6) % row-matrix from 3 to 4 with 6 steps

Special matrices are those with zeros, ones, and the unit matrix:
x = zeros(3) % create and show a (3,3)-matrix withs zeros
3+zeros(2,4) % show the result of a scalar addition to a (2,4)-matrix
y = 5*ones(3,2) % create and show a (3,2)-matrix with ones and multiply with 5
[x,y;y,x] % show a (5,5)-matrix composed from modules
eye(4) % show a unit matrix of size (4,4)

Extract the diagonal elements with diag:
diag(eye(3)) % show the diagonal elements of a (3,3)-unit matrix

Reshape or unravel a matrix with:
x = [1 3 2; 5 8 6]; % create a (2,3)-matrix
reshape(x,3,2) % use its elements in a (3,2)-matrix
x(:) % create a column-matrix using its elements

Products, divisions and powers are defined as matrix operators; the product of matrices A and B is a new matrix C with elements ci,j such that ci,j = Σk ai,k bk,j. The number of columns in A must, therefore, equal the number of rows in B. Direct products (ci,j = ai,j bi,j) have an extra dot. The inverse of a matrix is obtained with inv; this is matrix such that multiplication with the matrix results in the unit-matrix.
a = [3 5;3 1]; % create a (2,2)-matrix
b = inv(a); % calculate its inverse
a*b % matrix product
a.*b % direct product
a^2 % matrix power
a.^2 % direct power

Transpose (i.e. interchange rows and colums) with ':
a = [1;5;6]; b = [2 3 0]'; % create two column-matrices
a'*b % inner product, which is a (1,1)-matrix
a*b' % outer product, which is a (3,3)-matrix

Frequently used operators are max, min, sum, prod, sort, linear systems are solved with operator \ (see Octave's manual). Constants with a known value are e, pi (which can be overwritten). Well-known functions are sin, log, exp. Conversions from characters to numbers and vice versa are done with num2str, str2num. Some examples:
a = [1 3 5; 2 4 3]; % assign a (2,3)-matrix to a variable
sum(a) % sum across dimension 1, which is the rows; also: sum(a,1)
sum(a,2) % sum across dimension 2, which is the columns
prod(a) % multiply across dimension 1
max(a,4) % take the maximum of each element of a and 4
[am,ind] = max(a) % assign maximum element of a across rows for each column and the row indices for which the maxima are reached

Long lines are cut like
x = 2.1/ (4^2 + 3 * log(2)) + ... % calculate using 2 lines of text
exp(-7) * pi

If closing brackets are missing, Octave catenates new lines automatically.

Erase variables variable x with clearx, erase all variables with clear.

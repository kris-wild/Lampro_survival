addCV

appends parameter weights and cv's to addCV.html

Contents

  • Syntax
  • Description
  • Remark

Syntax

addCV

Description

Multi-species parameter estimation, using pars_init_group and run_group makes use of augmented loss functions that also minimize variation coefficients of specied parmeters. This function adds a row to a report-table (or initiates such a table, with mean MRE, SMSE and weights and cv for the various parameters. The system-browser is automatically opened if the table is initiated. First apply run_group to convergence, and give command addCV to append a new report-row and hit refresh in the browser. The information is taken from metaPar in results_group.mat.

Remark

The value of the loss function does not include contributions from the augmented term or pseudo-data

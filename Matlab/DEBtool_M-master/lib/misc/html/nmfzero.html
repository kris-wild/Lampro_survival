nmfzero

Solves a uni-variate function using Nelder Mead's simplex method

  • Description
  • Remarks
  • Example of use

Syntax

[q, info] = nmfzero (hfunc, p, varargin)

Description

Solves a uni-variate function using Nelder Mead's simplex method

Input

  • hfunc: function handle of user-defined function
   f = func (p, varargin) with
   p: initial value for variable that is to be solved
   value = func (p, varargin)
  • p: scalar
  • options: optional structure with parameters setting of the numerical procedure
  report, max_step_number, max_fun_evals, tol_simplex, tol_fun
  • varargin: optional parameters

Output

  • q: scalar: with solution
  • info: 1 if convergence has been successful; 0 otherwise

Remarks

Calls user-defined function 'func' Set options with nmregr_options Similar to fzero, but here using simplex method

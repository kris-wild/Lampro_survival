ddfnfourier

computes second derivatives of a function

Contents

  • Syntax
  • Description
  • Remarks
  • Example of use

Syntax

ddf = ddfnfourier(t, p)

Description

computes second derivatives of a function specified by n fourier coefficients

Input

  • t: (nt,1)-vector with time points
  • p: (n+1,2)-matrix with parameters
     - p(1,:): period, mean level
     - p(2:n+1,:): fourier coefficients for cosinus, sinus

Output

  • ddf: (nt,1)-vector with second derivatives of function

Remarks

Input-output structure similar to spline; cf dfnfourier for derivative; ifnfourier for integration; rfnfourier for roots; get_fnfourier for parameters; efnfourier for local extremes.

Example of use

see mydata_smooth.

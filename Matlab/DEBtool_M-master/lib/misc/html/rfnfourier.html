rfnfourier

calculates real roots t of fnfourier(t,p) = y on interval (0,period)

Syntax

f = rfnfourier(t, p)

Description

calculates real roots t of fnfourier(t,p) = y on interval (0,period)

Input:

  • ty: (r,2)-matrix with parameters; r>3
  • y: scalar with function value (optional, default is 0)

Output:

  • x: vector with real roots

Remarks

Input-output structure similar to spline; cf dfnfourier for derivative; ifnfourier for integration; rfnfourier for roots; get_fnfourier for parameters; efnfourier for local extremes.

Example of use

see mydata_smooth.
